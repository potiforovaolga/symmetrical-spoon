Список библиотек, используемых в Android Studio 2024: 
▎Язык программирования - Kotlin применяется для разработки Android-приложений. 
▎1. Jetpack Compose: Набор инструментов для создания нативных пользовательских интерфейсов с использованием декларативного подхода.
▎Библиотеки Android Jetpack
• ViewModel: Позволяет сохранять UI-состояние при изменении конфигурации (например, при повороте экрана).
• Room: Библиотека для работы с SQLite базами данных, обеспечивающая удобный доступ к данным с использованием аннотаций.
• DataStore: Современное решение для хранения пар ключ-значение и сериализованных данных.
• WorkManager: Управление фоновыми задачами, которые должны выполняться даже если приложение закрыто.
• Navigation: Упрощает навигацию между экранами приложения, поддерживая граф навигации.
• CameraX: Упрощает работу с камерой устройства, обеспечивая совместимость с различными устройствами.
• Media3: Библиотека для работы с мультимедийным контентом, включая потоковую передачу и локальные медиафайлы.
• Glance: Библиотека для создания виджетов и взаимодействия с ними.
 
▎2. Material Design 3
• Material You: Новый подход к кастомизации интерфейса, который позволяет пользователям изменять внешний вид приложения в соответствии с их предпочтениями.
• Material Theme Builder: Инструмент для создания тем приложения с использованием принципов Material Design.
▎3. API SplashScreen: Встроенный API для управления экранами загрузки в приложениях на Android 12 и выше, позволяющий создавать анимации и настраиваемые экраны.
▎4. MVVM и MVI:Для реализации MVVM и MVI в Android используются различные библиотеки, которые помогают организовать код и управлять состоянием пользовательского интерфейса.
▎MVVM
 • Android Architecture Components:
   - ViewModel: Позволяет хранить и управлять UI-ориентированными данными, сохраняя их при изменении конфигурации.
   - LiveData: Объект, который может наблюдать за изменениями данных и автоматически обновлять UI.
   Эти компоненты входят в состав библиотеки androidx.lifecycle.
• Data Binding (опционально): Позволяет связывать UI-компоненты с данными из ViewModel, что упрощает реализацию двусторонней привязки.
▎MVI
• MVI Libraries: MviKotlin: Библиотека для реализации паттерна MVI на Kotlin, предоставляющая инструменты для управления состоянием и интентами.
• Coroutines (опционально): Часто используются для асинхронного программирования в MVI.
▎5. Инъекция зависимостей
• Hilt: Библиотека для внедрения зависимостей, основанная на Dagger, которая упрощает настройку DI в Android-приложениях.
• Dagger: Фреймворк для инъекции зависимостей, предоставляющий инструменты для управления зависимостями.
• Koin: Легковесная библиотека для инъекции зависимостей на Kotlin, которая предлагает простой и понятный синтаксис.
▎6. Сетевые библиотеки
• Retrofit: Библиотека для работы с RESTful API, упрощающая выполнение сетевых запросов и обработку ответов.
• OkHttp: HTTP-клиент для Android и Java, используемый в связке с Retrofit.
• Ktor: Асинхронный фреймворк для создания клиентских и серверных приложений на Kotlin.
▎7. Сериализация
• Moshi: Библиотека для сериализации JSON, которая хорошо работает с Retrofit.
• Kotlin Serialization: Официальная библиотека для сериализации данных в различных форматах.
▎8. Загрузка изображений
• Coil: Легковесная библиотека для загрузки изображений с поддержкой корутин.
• Glide: Библиотека для загрузки изображений с мощными возможностями кэширования и обработки изображений.
▎9. Реактивность / управление потоками
• Kotlin Coroutines: Поддержка асинхронного программирования с использованием корутин и Flow.
• RxJava: Популярный инструмент для реактивного программирования в Android.
▎10. Локальное хранилище
• DataStore: Современное решение для хранения данных с поддержкой асинхронных операций и реактивного программирования.
• EncryptedSharedPreferences: Позволяет хранить данные в зашифрованном виде.
▎11. Тестирование
• JUnit 5: Основной фреймворк для юнит-тестирования на Java и Kotlin.
• MockK: Библиотека для создания моков в Kotlin.
• Espresso: Инструмент для UI-тестирования приложений на Android.
• Robolectric: Позволяет запускать тесты Android в обычной JVM.
▎12. Тестирование скриншотами
• Shot: Библиотека для автоматизированного тестирования визуальных изменений в приложениях через скриншоты.
Это неполный список.
